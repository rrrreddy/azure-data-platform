# Azure Data Platform

Production-grade Azure Databricks platform featuring:
- **Mortgages Real-Time Analytics**: Real-time mortgage application processing
- **Employee Performance Analytics**: Time allocation and productivity metrics

## Quick Start
```bash
git clone ...
cd azure-data-platform
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
